## 🚀 Featured Projects

### [Designer Console](https://github.com/tristan-wright-program-extraordinaire/designer-console)
*InDesign plug-in — JavaScript / React*

A fully-featured InDesign plug-in that connects a CRM and network drive directly into the tool designers live in all day. Eliminates context-switching by surfacing advertiser data, file management, and production tools inside the creative environment. Measurably doubled designer efficiency across the team.

---

### [Sales Floor Portal](https://github.com/tristan-wright-program-extraordinaire/sales-floor-portal)
*Django · React · PostgreSQL · Google Cloud Run*

A production platform for sales reps, managers, and admins to manage course inventory and sourcing. Handles continuous automated merging of two independent course databases, role-based auth, and a frontend with built-in lead sourcing tools. Deployed and actively used on Google Cloud Run.

---

### [Course Maps](https://github.com/tristan-wright-program-extraordinaire/course-maps)
*React · Google Maps API · Python · Google Cloud Run*

An interactive nationwide map that displays all signed courses with product-level detail for sales representatives in the field. A daily Python pipeline keeps course data fresh automatically. Hosted on Google Cloud Run.

---

### [CS Workflow App](https://github.com/tristan-wright-program-extraordinaire/cs-workflow)
*Python · PyWebview · React*

A desktop app that pulls a dynamic call flow chart from Google Sheets and renders it as a live, interactive interface for customer service reps. The CS manager controls the flow from a spreadsheet; the reps see it update in real time.

---

### [Design Proof Auditing App](https://github.com/tristan-wright-program-extraordinaire/design-proof-auditing)
*Python · PyWebview · React*

A standalone desktop audit tool for designers. Loads graphic proofs from a zip file for review, then routes approved or rejected files to the right destination automatically. Reduces the need for manual file management to almost zero.

---

### [Hex TD Game *(active personal project)*](https://github.com/tristan-wright-program-extraordinaire/Vire-TD-Game)
*Godot*

A roguelike tower defense played on a hex grid. Your base sits at the center; enemy nests spawn on the outer edges. Towers don't just block — they shape the path. Where you place them determines the routes enemies can take, making every decision spatial and strategic. The roguelike layer keeps runs dynamic and constantly evolving. Core mechanics are built out; content and progression systems are in active development.
